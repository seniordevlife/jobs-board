# BulletinBoard
Simple job market portal CRUD with user roles.

## Architecture
- MVC
- ViewModel pattern
- Repository / Service pattern
- Dependency Injection
- Entity - Model mapping

## Frameworks and tools
- ASP.NET Core MVC
- ORM: EntityFramework Core
- IoC container: Autofac
- Automapper

## Tests
- xUnit with Moq
- Selenium + Coded UI
